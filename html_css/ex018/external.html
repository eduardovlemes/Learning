Exemplo com fontes baixadas

Caso você tenha baixado uma fonte no pc e queira usar no html, basta usar o @font-face para indicar de onde virá a fonte. A maioria dos navegadores trabalham com documentos de fonte do tipo .otf (format usado é o 'opentype') e .ttf (format usado é o 'truetype').
Exemplo do código:

        
            @font-face {
                font-family: 'bg';
                src: url('fonts/bg.ttf') format('truetype');
            }
            body{
                font-family: 'bg';
                font-size: 40px;
            }
