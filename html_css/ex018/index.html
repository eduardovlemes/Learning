Exemplo - Google Fonts

Nesse exemplo temos dois tipos de fontes. A que está no h1 é "permanent marker", e a que está no parágrafo é a "raleway". Ambas as fontes foram importadas do site do google fonts.
Para isso, na tag <style>, imediatamente abaixo da tag, é inserido o código copiado do google fonts e, posteriormente, mencionado no font-family de <h1> e o outro no font-family de <p>. 

        
            style
                @import url('https://fonts.googleapis.com/css2?family=Permanent+Marker&display=swap');
                @import url('https://fonts.googleapis.com/css2?family=Raleway&display=swap');
                h1{
                    font-family: 'Permanent Marker';
                }
                p{
                    font-family: 'Raleway';
                }
            /style
